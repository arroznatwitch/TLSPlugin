# The Last Survivor - Plugin

Paper 1.21+ plugin that runs the *The Last Survivor* event. 

Website: [playthelastsurvivor.com](https://playthelastsurvivor.com)

## Depends on

- Paper 1.21+, Java 21
- ItemsAdder, ProtocolLib, packetevents (required)
- LuckPerms (optional)

## Build

```bash
git clone https://github.com/arroznatwitch/TLSPlugin.git
cd TLSPlugin
./gradlew build
```
